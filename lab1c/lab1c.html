Lab 1c - Pacman

Camera

Toggled by pressing the spacebar. Controlled by using the arrow keys and , (forward movement) and . (backward movement) or dragging the scene with the mouse. Initially disabled.

Light Movement enabled: false

Camera Movement enabled: false

Shading
  • w Gouraud, diffuse and ambient only
  • e Gouraud, Phong illumination model (+ specular)
  • r Phong, diffuse and ambient only
  • t Phong, Phong illumination model (+ specular)
Objects

Selected by pressing 1-9, 0 selects all objects. All transformations only apply for selected objects.

Translation

The translation is applied to the selection by using the arrow keys and ',' for forward translation and '.' for backward translation as long as camera movement is disabled
